AngularJS Modules

An AngularJS module defines an application

The module is a container for the different parts of an application

The modile is a container for the application controllers

Controllers always belong to a module

Creating a Module

A module is created by using the AngularJS function angular.module

var app = angular.module("myApp", []);

The "myApp" parameter reders to an HTML element in which the application will run

You can add controllers, directive, filters, and more

Adding a Controller

Add a controller to your application with the ng-controller

Controller vars: {{ firstName + " " + lastName }}

Adding a Directive

AngularJS has a set of built-in directives which you can use to add functionality to your application

You can use the module to add your own directives to your applications

The HTML directive attribute use dash to separate words and no camel-case eg: my-directive

The directive in the module uses camel-case eg. myDirective
