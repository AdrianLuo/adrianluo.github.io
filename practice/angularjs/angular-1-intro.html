AngularJS is a JS Framework

AngularJS is a JS framework which can be added an HTML page.

AngularJS extends HTML attributes with directives, and binds data to HTML with expressions.

The framework can be included using a script tag to call: https://ajax.googleapis.com/ajax/libs/angularjs/1.6.9/angular.min.js

AngularJS Extends HTML

AngularJS extends HTML with ng-directives

The ng-app directive degines an aJS application

The ng-model directive binds the value of HTML controls (input, select, textarea) to application data

The ng-bind directive binds application data to the HTML view

AngularJS Expression

Expressions are written inside double braces { { expression } }

Expressions can also be written inside a directive: ng-bind="expression"

AngularJS will resolve the expression, and return the result exactly where the expression is written

AngularJS expressions are much like JavaScript Expressions: they contain literals, operators and variables { { 5 + 5 } } or { { firstName + " " + lastName } }

My expression: 5 + 5 = {{ 5 + 5 }}

Example of changing css properties with an initialized variable: ng-init="myCol='#888'"
